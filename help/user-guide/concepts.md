# 概念

GenStudio for Performance マーケターは、Adobeのコンテンツサプライチェーンを具体化してマーケティングキャンペーンを合理化するスタンドアロンのエンタープライズ製品です。 パーソナライズされ、ブランドで承認されたコンテンツを大規模に構築し、有効性を監視し、常に変化する市場に迅速に適応することは困難です。 GenStudio for Performance マーケターは、ジェネレーティブ AI をエンタープライズマーケティングチームのパフォーマンスの乗数として活用する、1 つのアプリケーションにCreative CloudとExperience Cloudをもたらします。

パフォーマンスマーケター向けGenStudioを使用すると、次のことができます。

- 自然言語プロンプトを使用したオンブランドコンテンツの作成
- 関係者との共同作業による、生成されたコンテンツのレビューと承認
- 生成および承認されたコンテンツを保存して、今後のマーケティングキャンペーン用にアクセスできる
- キャンペーンとコンテンツ属性に関するインサイトでコンテンツのパフォーマンスを評価する

## ジェネレーティブ AI テクノロジー

GenStudio for Performance マーケターは、ジェネレーティブ AI の力を活用して、コンテンツ作成プロセスを加速し、高品質なコンテンツ生成を確実に行います。 クリエイティブアセットの反復的なライフサイクルにより、ターゲットオーディエンスの共感を得られる、より正確でブランドに合わせたコンテンツが得られます。

強力なブランドガイドライン機能を使用して、組織のブランディング、顧客のペルソナ、製品説明について、パフォーマンスマーケター向けにGenStudioを効果的にトレーニングすることから始めます。 これらのガイドラインの準備とアップロード方法については、[ ガイドラインの概要 ](../user-guide/guidelines/overview.md) を参照してください。

## コンテンツのライフサイクル

複数のチャネルで、より速いペースで質の高いエクスペリエンスが求められています。 GenStudio for Performance マーケターは、コンテンツサプライチェーンを、マーケター向けのよく整理されたワークフローに簡素化します。 GenStudio for Performance マーケターは、ライフサイクルの各段階でAdobeテクノロジーを活用しています。
